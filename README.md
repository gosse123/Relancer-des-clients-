# Relance automatique de clients vias n8n
## ce workflow vous permet de relancer aoutomatiquement des clients vias n8n 

# Prerequis
 - Avoir un compte charriow
 - Heberger n8n en local ou en ligne
 - Avoir un compte gmaile pour l'envoir des mail au clients

# Installation et Configuration 
  - telecharger ou copier le contenue du fichier "doc/main.json"
  - creer un nouveau workflow sur n8n
  - Importer le fichier ou coller le contenue du fichier "doc/main.json"
  - configurer les api dans le noeud gmail
